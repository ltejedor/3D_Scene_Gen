# Development Setup

## Running the Application

### Backend
```bash
cd backend
uvicorn app:app --reload
```

### Frontend
```bash
cd frontend
python -m http.server 5500
```

Download objects folder from https://drive.google.com/drive/folders/1RQdtVHAYEXV7NqsdnskbWSXSdtD72LFi?usp=drive_link
